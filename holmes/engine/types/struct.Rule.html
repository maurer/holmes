pub struct Rule {
    pub name: String,
    pub head: Clause,
    pub body: Vec<Clause>,
    pub wheres: Vec<WhereClause>,
}

A Rule represents a complete inference technique in the Holmes system If the body clauses match, the wheres clauses are run on the answer set, producing a new answer set, and the head clause is instantiated at that answer set and inserted into the database.

Fields

name: String

Identifier for the rule

Template for the facts this rule will output

Datalog body to search the database with
